URL Shortener microservice

How does this work?

  1. Call the /new api followed by the url of the site you wish to shorten.

  2. If the url is valid, this service will return a JSON object that includes both the requested url and the shortened url.

  3. However, if the url is invalid, then it will return a JSON object with an error message.

    For Instance:

    Input ->

    https://follow.glitch.me/new/http://helloenjoy.com/project/lights/

    Output ->

    { "original_url" : "http://helloenjoy.com/project/lights/" , "short_url" : "https://follow.glitch.me/HJd8hPMDW" }

    See it in action by clicking on this link: https://follow.glitch.me/HJd8hPMDW
